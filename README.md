#  Virtual Pen

A browser-based drawing app where you can draw in the air using your finger.

## Features
- Draw using hand gestures
- Real-time hand tracking
- Smooth drawing effect
- Press SHIFT to draw
- Press SPACE to clear canvas

## Tech Stack
- HTML, JavaScript
- MediaPipe Hands
- Canvas API

## Future Improvements
- Color selection
- Save drawing as image
- UI enhancements
